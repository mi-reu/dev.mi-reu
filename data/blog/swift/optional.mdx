오늘 정리할 내용은 스위프트의 `Optional` 입니다 👏👏👏

스위프트의 옵셔널은 일종의 타입인데요. 정확하게는 Enum 타입입니다.<br />
Enum 은 열거형 타입이라고 하고 아래와 같이 `enum` 키워드로 정의된 타입이 Enum 타입 입니다.

```swift
enum Optional<Wrapped> {
    case none // == nil
    case some(Wrapped) // value
}
```
